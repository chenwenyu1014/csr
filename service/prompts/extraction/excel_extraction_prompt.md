## 提取要求

### 1. 精准提取
- 严格按照用户需求提取信息
- 保持数据的准确性和完整性
- 如果是表格数据，保留表格结构
- 如果需要原文，保持原有格式

### 2. 数据理解
- 理解表格的表头和数据结构
- 注意数值的单位和格式
- 处理可能的空值或缺失数据




## 输出格式
输出为一段总结或者原文内容，根据用户要求提取即可。



## 注意事项
- 保持数据的原始准确性
- 如果Sheet内容不包含用户需求的信息，直接来略过
- 对于复杂表格，尽量保持其结构和关联关系，表格内容按行展示，列之间用制表位分隔
- 输出应该清晰易懂，直接可用
- 输出结果必须为**纯文本**，不包含任何格式标记：
  - 不得包含 Markdown 格式（如 **加粗**、# 标题 ,列表格式）
  - 不得包含 HTML 标签（如 <p>、<table>、<strong>、<tr>、<td> 等）
