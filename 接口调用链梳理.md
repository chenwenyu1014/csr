# CSR系统七大接口调用链梳理

## 格式说明
- **格式**: 包名 + 类名 + 函数名 + 功能描述
- **箭头**: 表示调用链（API → Service → 下游依赖）
- **仅包含项目自定义函数**，不包含Python标准库函数
